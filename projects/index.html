1. Agent Based Modeling


skills
Coding (R), data visualization

Agent based modeling is a computer simulated environment in which agents interact with other agents within an environment defined by manipuable parameters. This example delves into the schelling-segregation-model, where the purpose was to determine whether segregation can be explained by individual motivations to avoid being a local minority. Agents move around a two-dimensional space depending on whether they are a racial minority within their 'local neighborhood'.
