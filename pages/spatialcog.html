WORK ABOUT
Research: Spatial Experiences

Researching how spatial experiences like hiking or computer programming, can impact performance in tasks like navigation or mentally rotating objects (like chemical molecules)

Skills

User research, Data Visualization and Analyses

Tools

R, immersive Virtual reality, desktop Virtual Reality, Psychopy, Python, Excel

I spent my time in the Spatial Thinking Lab researching how spatial experiences impacting performance in immersive Virtual Reality & desktop tasks. I also worked as a lead on an independent project on spatial activities, on creating reproducible scripts for data analysis, administering cognitive desktop tasks and surveys, and acting as the lab manager by helping train new research assistants.

Spatial abilities can be developed by training and everyday experience, but we know little about how different types of experiences are related to specific spatial competencies. Does using maps while hiking improve skill in navigation? Does playing video games improve the same (i.e. navigation) or different (e.g., mental rotation) abilities? We were also interested in how the interface of the test interacted with people's performance and experience.

Would there be a familiarity with the controls of desktop tasks (computer) as opposed to immersive virtual reality?

Challenges and Learnings

Be specific and concise

Working on manuscripts and writing procedures has helped me understand how to use fewer words to explain our experimental methods. Telling a story should be gripping yet effective!

Reproducible code

I wrote adjusted code in R to process data from a Mental Rotation Task from a large dataset. This taught me how to ensure that my own code should be accessible (using R Markdown), creative and efficient (understanding which commands to use), and ask for help when I needed to, by refering to more experienced coders to create the best script possible.

Choosing the right method for the research question

Whether it was choosing the statistical method or choosing an exclusion criteria, the method should be supported by prior research in mind and/or discussed with multiple members on the team.
